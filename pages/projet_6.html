Quiz'actu

Le Projet

Quiz'Actu est un jeu en ligne de questions d’actualité. Chaque jour, un challenge de 10 questions sous forme de QCM est proposé aux joueurs. Ces questions concernent l’actualité du jour. Avec la réponse, le lien d’un article traitant cette actualité apparaît pour permettre aux joueurs, s’ils le souhaitent, d’en savoir davantage sur le sujet.

Objectifs

Le but de ce projet était de créer une application web et mobile Java Jee en un mois qui permettrait de réaliser le projet de Agathe Perrier et Loïs Elziere, tous deux journalistes à Marseille depuis trois ans. Grâce a notre formateur Jeremy Masson et toute mon équipe nous avons pu réaliser a bien ce projet.

Logiciels

Photoshop, Eclipse, StarUML, Workbench, Gitkraken, Méthodes agiles

Langages

JAVA JEE - Spring-boot - Thymeleaf - CSS - Boostrap
