Agence Immo

Le Projet

Agence Immo est une application web qui permet de mettre en ligne des biens en ligne à vendre. Ce projet a été réalisé dans le cadre d'une formation Udemy.

Objectifs

Mes objectifs étaient à la fois de pouvoir d'apprendre de nouveaux concepte sur Angular 8, une Framework que j'apprécie particulièrement. Comprendre mieux le concepte de MVC, ansi que toutes les liaison de données entre le template et le typescript. J'ai pu également découvrir le deployement sur Firebase.

Logiciels

VScode - Firebase

Langages

Angular 8 - HTML & CSS - Bootstrap - JQUERY
